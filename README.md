# 🎮 Reflex Rush: Two-Player Arduino Reaction Game

Reflex Rush is a fun two-player reaction game built using an Arduino Uno. Players compete to see who can react the fastest by pressing their assigned button when the signal appears. LEDs provide visual cues, while a buzzer adds sound effects to make the game more exciting.

## ✨ Features

- Two-player gameplay
- Fast-paced reaction challenge
- LED visual indicators
- Push button controls
- Buzzer sound effects
- Easy to build and beginner-friendly

## 🛠️ Components Used

- Arduino Uno
- Breadboard
- 3 LEDs
- 2 Push Buttons
- Piezo Buzzer
- 220Ω Resistors
- Jumper Wires

## 🎮 How to Play

1. Two players each take control of one push button.
2. Press the start button to begin the game.
3. Wait for the signal LED to turn off.
4. As soon as the game starts, both players race to press their button.
5. The player with the fastest reaction wins the round.
6. LEDs and the buzzer indicate the winner before the next round begins.
